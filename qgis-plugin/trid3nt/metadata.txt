about=Dockable chat panel that drives a TRID3NT agent (local stack on ws://127.0.0.1:8765 or a remote endpoint). Every layer the agent publishes is materialized as a native QGIS layer: rasters as XYZ tile layers (TiTiler templates), vectors from inline GeoJSON or the local MinIO object store. Milestone 3 of v1: simulation gate card, canvas-extent or selected-polygon AOI on every message, capped-jitter auto-reconnect with queued sends, case switching (open any existing case's chat) with a debounced case-list refresh, Open-case-in-QGIS in BOTH modes (local export API; remote downloads the .gpkg/.qgz artifacts), and honest token-expiry handling in remote mode. The local TRID3NT stack (agent, MinIO, TiTiler, Ollama) must be installed and running separately -- the plugin never starts or stops it.
